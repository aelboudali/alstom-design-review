# New in Industry Viewer Template 2.0

Industry Viewer Template 2.0 introduces several new features and improvements to enhance the user and developer experience.

## Add local assets to the Industry Viewer Template

You can now add local 3D assets directly to the Industry Viewer Template without needing to upload them to Unity Cloud. This feature allows for quicker testing and iteration during development. For more information, refer to [Understand the login and asset selection process](get-started/login-and-asset-selection.md#add-a-new-asset).

## Asset comments and collaboration

You can now add comments to assets directly within the Industry Viewer Template. Users can collaborate by leaving feedback, attaching files, and viewing comment threads in real time. This feature syncs with Unity Asset Manager to ensure all users have access to the latest discussions. For more information, refer to [Understand the login and asset selection process](get-started/login-and-asset-selection.md#add-comments-to-assets).

You can also add annotations directly onto 3D assets in the scene, making it easier to highlight specific areas of interest or concern during reviews.

## Toolbox enhancements

A new **Toolbox** option is available to users, providing access to additional tools for interacting with 3D assets. The toolbox includes:

* Measurement tool: Measure distances between points in the 3D scene.
* Cross-section tool: Create cross-sectional views of 3D assets for better internal visualization.
* Change environments: Swap between different environments to view assets in various settings and lighting conditions.

For more information, refer to [Explore the Tool Options menu](get-started/tool-options.md).

## Link sharing

You can now generate shareable links to specific assets or scenes within the Industry Viewer Template. This feature allows users to share their work with colleagues or clients for review and collaboration. For more information, refer to [in-app-controls-and-navigation.md](get-started/in-app-controls-and-navigation.md#share).

## Wireframe mode

A new wireframe mode has been added, allowing users to view a wireframe representation of 3D assets. This mode is useful for analyzing the geometry and structure of assets. For more information, refer to [Wireframe mode](project-settings-customization/feature-management/wireframe-mode.md).

## XR Passthrough support

The Industry Viewer Template now supports XR Passthrough, enabling users to view 3D assets in augmented reality using compatible XR devices. This feature enhances the immersive experience and allows for better spatial understanding of assets. For more information, refer to [In-app controls and navigation](get-started/in-app-controls-and-navigation.md#xr-support).
